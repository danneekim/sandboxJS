What is Javascript?

JavaScript was originally developed to run only in browsers.

  • JavaScript was originally developed to run only in browsers.
  • Every browser has what we call a JavaScript engine.
  • IE: Chakra / FireFox: SpiderMonkey / Chrome: v8

  • In 2008 - Ryan Dahl took the open source JavaScript Engine
  • and embedded it inside a C++ program - Node.exe

  • Node allows JavaScript code to run outside the browser.
  • Allowing JavaScript to be used to build the backend for our web/mobile applications.
  • JavaScript can be run on the browser or on Node.

JavaScript vs ECMAScript?

  • JavaScript - programming language
  • ECMAScript - specification

  • ECMA - organization resp. for defining JavaScript standards
  • The first version of ECMAScript was released in 1997.
  • Since 2015 (ES2015/ES6) - ECMA has released annual specifications.
